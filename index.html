自民党

項目 説明
経済・財政 物価高騰対策給付金（子ども・非課税世帯4万円、その他2万円）をマイナンバー活用で迅速給付。賃金上昇・GDP増、中小企業支援、NISA・iDeCoで資産形成促進、税制見直しを推進します。
社会保障 全働く人の所得増（正規・非正規格差是正、最低賃金引き上げ）、就職氷河期支援を進める。医療・介護・福祉の報酬改定で賃上げ、厚生年金適用拡大と基礎年金底上げで年金拡充を目指します。
農政 農業の生産性向上のため、既存予算とは別枠で思い切った予算を確保。コメの安定供給と円滑な流通確保へ事前契約を推進し、官民で総合的な備蓄体制を確立。水田政策も見直します。
外交・安保 日米同盟を基軸に同志国と連携し、防衛力を抜本的に強化して国民の命と領土を守り抜く。違法外国人ゼロ推進、憲法改正を通じて「国のかたち」を創り上げることを目指します。
エネルギー ガソリン暫定税率廃止については、財源・流通影響を踏まえ自動車関係税制全体の見直しと併せて議論を進める。2050年カーボンニュートラル実現へ地球温暖化対策計画を着実に実行します。
家族・子育て 高校授業料実質無償化など教育費負担軽減や学び直し支援を推進。児童手当の抜本拡充、男性育休促進、住宅支援など、多様な子育て支援メニューを十分に活用できる環境を整備します。
政治とカネ・政治改革 政治資金の透明化と厳正なコンプライアンスをいっそう推進する。政党のガバナンス体制強化のため、党ガバナンスコードに基づく改革のPDCAサイクルを回し続けます。

公明党

項目 説明
経済・財政 物価高騰対策として生活応援給付（1人2万円、子ども・住民税非課税世帯は4万円）を給付する。自動車関連税の引き下げや所得税の負担軽減、ガソリン暫定税率廃止時期の年末協議、奨学金返済額の所得控除検討を進めます。
社会保障 基礎年金の給付水準を底上げします。低所得者や子育て世帯向けに賃貸住宅の家賃を補助する「住宅手当」の創設を目指します。
農政 コメの安定供給確保のため増産し、流通実態の調査と取引適正化に向けた監督機能の強化を推進します。
外交・安保 人間の安全保障を基軸に対話外交を進め、北東アジア安全保障対話・協力機構を創設します。ウクライナの復旧・復興に向けた地雷除去支援などを主導します。
エネルギー 2050年カーボンニュートラル実現に向けた取り組みを推進し、日本の脱炭素技術を海外に展開します。
家族・子育て 妊娠・出産に関する基礎的な費用を無償化し、産後ケアを充実させます。「こども誰でも通園制度」の人材不足や利用時間などの課題解消を図ります。
政治とカネ・政治改革 企業・団体献金の規制強化については、橋渡し役として幅広い合意形成を目指します。

立憲民主党

項目 説明
経済・財政 物価高対策として食料品消費税ゼロ（1年間）、国民へ2万円給付、「給付付き税額控除」を実施。ガソリン暫定税率廃止、円安是正に向けた政府・日銀共同声明見直し、先端技術開発の国家プロジェクト推進を図る。
社会保障 最低賃金を早期に全国一律1500円以上へ引き上げ、「年収130万円の壁」を給付で解消。医療・介護・保育など国民生活を支える職種の処遇改善、同一労働同一賃金、インボイス制度廃止、就職氷河期支援を推進する。
農政 コメ政策を見直し、備蓄米制度や生産体制を抜本強化する。食料と農地を守るため、農家への直接支払制度を創設し、就農支援資金を10倍に増額することで、持続可能な農業を目指す。
外交・安保 憲法の平和主義に基づき安保法制の違憲部分を廃止し、専守防衛の外交・安保政策を構築。辺野古移設中止、日米地位協定見直し要求。防衛増税はせず自衛隊員処遇改善、核廃絶へ働きかける。
エネルギー 2050年までの早期に化石燃料・原発に依存しないカーボンニュートラル達成を目指す。原発の新増設は認めず、廃炉作業を国管理下に。省エネ・再エネへ10年で200兆円投資、建物断熱化を推進する。
家族・子育て 公立小中学校給食費、国公立大学授業料の無償化、私立も同等に負担軽減。就学前教育・保育無償化。18歳まで月1.5万円の児童手当、奨学金負担軽減。選択的夫婦別姓・同性婚の早期法制化を目指す。
政治とカネ・政治改革 企業・団体献金を全面禁止し、裏金を許さず政治資金の徹底的な透明化を図る。国会議員の政治資金の世襲を制限し、政治資金パーティーの企業・団体による購入も禁止し収入公開を強化する。

日本維新の会

項目 説明
経済・財政 食料品の消費税を2年間ゼロとし、現役世代の勤労所得税額控除や給付付き税額控除を導入。就職氷河期世代の安定雇用と能力開発も支援し、経済を活性化する方針です。
社会保障 国民医療費を年間4兆円以上削減し、現役世代の社会保険料を年間6万円引き下げる。OTC類似薬の保険適用除外や電子カルテ普及100%達成で医療制度の効率化をめざす。
農政 「抑える農政から伸ばす農政」へ転換し、稼げる農業をめざす。多様な主体による農業への参入を支援することで、持続可能な農業体制を構築し、担い手不足の解消を図る。
外交・安保 国民負担増なしに防衛費をGDP比2%へ増額し、「積極防衛能力」を整備して国民の生命と財産を守る。貿易ルール順守国と集団での経済安全保障体制を構築する。
エネルギー 規制改革と投資促進を通じて、グリーントランスフォーメーション（GX）を強力に推進する。持続可能な社会実現に向けたエネルギー政策を主導します。
家族・子育て 進学しなかった入学金の返還制度を導入。大学改革後に授業料無償化を段階的に拡充し、未就学児の教育・保育は所得制限のない完全無償化をめざし、子育て・教育費負担を軽減する。
政治とカネ・政治改革 国会議員の歳費と定数を3割カットし、政治家の身を切る改革を実行する。企業・団体からの献金は一切受け取らないことで、政治とカネの問題の根絶を目指す。

共産党

項目 説明
経済・財政 消費税の廃止を目指し、まずは5%へ減税します。恒久財源確保のため、大企業の法人税と富裕層の所得税・住民税・相続税の税率を引き上げます。最低賃金は時給1500円（手取り月20万円）に早期に引き上げ、1700円を目指し、労働時間は「1日7時間、週35時間制」への移行を国の目標とします。
社会保障 年金支給額の引き上げのため、マクロ経済スライドを直ちに撤廃します。医療分野では、国費5000億円を投入して診療報酬を引き上げ、患者負担増なしに医療従事者の賃上げを図ります。
農政 農家のコメ手取り60キロあたり2万〜2万数千円の保障と、農林水産省の予算を1兆円増額することを掲げています。コメは減反・減産から増産に転換し、備蓄米の買い入れ量を計画的に増やし200万トン以上を目指します。
外交・安保 沖縄の米軍新基地建設を中止し、日米地位協定の抜本改定を求めます。また、トランプ関税を許さず、無条件での全面撤回を強く要求します。
エネルギー 再生可能エネルギーを大胆に導入し、すみやかに原発ゼロを目指します。
家族・子育て 大学までの教育無償化を目指し、教育予算の拡充、学費値下げ、教職員の増員を推進します。家賃が所得の2割を超える人に対し、超過分の最大15%を減税する支援策を提案しています（高額所得者や高額家賃は除く）。選択的夫婦別姓制度と、同性婚を認める民法改正を実現します。
政治とカネ・政治改革 パーティー券購入を含む企業・団体からの献金を全面禁止することで、政治資金の透明化と健全化を図ります。

国民民主党

項目 説明
経済・財政 消費税を賃金上昇安定まで5%に減税し、年収の壁を178万円に引き上げ、30歳までの若者の所得税・住民税を減免します。ガソリン税の旧暫定税率も廃止する方針です。
社会保障 後期高齢者医療の窓口負担を原則2割とし、現役世代の社会保険料を削減します。基礎年金底上げのため国民年金追納要件の緩和も目指します。
農政 農家に対し、10アールあたり年1万～1万5000円の「食料安全保障基礎支払」を導入することで、農業の安定化を図ります。
外交・安保 「スパイ防止法」の制定を掲げ、外国人による不動産投資を規制することで、国の安全保障を強化する方針です。
エネルギー 再生可能エネルギー発電促進賦課金（再エネ賦課金）の徴収を停止し、安全基準を満たした原発の早期稼働と次世代炉開発を推進。エネルギー自給率50%達成を目指します。
家族・子育て 教育国債を年5兆円発行し子育て・教育予算を倍増します。0～2歳児の教育・保育無償化の所得制限を撤廃し、高校までの教育費を完全無償化します。
政治とカネ・政治改革 政治資金の規制強化と透明化を推進します。また、被選挙権年齢を18歳に引き下げ、インターネット投票の導入により政治参加を促します。

れいわ新選組

項目 説明
経済・財政 消費税を廃止し、現金10万円の一律給付や季節ごとの給付金で物価高に対応します。最低賃金を全国一律1500円に引き上げ、大企業・富裕層への課税強化で財源を確保します。
社会保障 後期高齢者医療制度を廃止し全額国庫負担とし、介護保険の国負担増で保険料を軽減。年金はマクロ経済スライドを廃止し、介護・保育職員の月給を10万円アップさせます。
農政 農林関係予算を倍増し5兆円に引き上げ、**食料自給率をまず50%**にします。余剰農産物は国が買い上げ、国内外の食料支援に活用します。
外交・安保 沖縄辺野古新基地建設を中止し、安保関連3文書の廃止・撤回を求めます。5年間43兆円の軍事費倍増計画も中止します。
エネルギー 原子力発電所は即時廃止し廃炉を推進します。2050年までのできるだけ早い時期に二酸化炭素（CO2）排出量を実質ゼロにすることを目指します。
家族・子育て 高校卒業まで子ども手当を月3万円（所得制限なし）支給し、大学院までの教育無償化を目指します。同性婚と選択的夫婦別姓の法制化も実現します。
政治とカネ・政治改革 供託金制度を廃止し、衆参の被選挙権年齢を18歳に引き下げます。政治参加のハードルを下げ、開かれた政治を目指します。

参政党

項目 説明
経済・財政 消費税の段階的廃止と社会保険料負担の軽減を掲げ、**国民負担率を上限35%**に設定します。また、外国人総合政策庁を新設し、外国人受け入れ基準や制度運用を一元管理し、外国人による土地・不動産購入に厳格な制限を設けます。
社会保障 健康維持と医療費削減に貢献した高齢者に対し、国内旅行で使えるクーポン券を支給する方針です。
農政 10兆円規模の予算を投じ、10年以内に食料自給率を倍増させることを目指します。
外交・安保 直接的な外交・安保の項目はないものの、外国人による土地・不動産購入に厳格な制限を設けることで、国の安全保障上の脅威に対応します。
エネルギー 参政党の主要公約において、エネルギー政策に関する具体的な記述は提供された情報からは見当たりません。
家族・子育て 偏差値重視の管理教育を廃止します。子ども1人につき月10万円の教育給付金を支給し、日本国籍を有する家庭を優先対象とします。
政治とカネ・政治改革 外国人参政権を一切認めず、帰化1世にも被選挙権を付与しないことを公約に掲げています。

社民党

項目 説明
経済・財政 食料品の消費税率を即時にゼロにし、財源は防衛費削減や法人税・所得税の累進性強化で対応します。
社会保障 すべての高齢者に月10万円を給付する最低保障年金制度の設計を急ぎます。また、最低賃金全国一律1500円の早期実現を目指します。
農政 農家の所得補償を実現し、食料自給率50%の達成を目指します。
外交・安保 沖縄の米軍普天間基地の辺野古への移設に反対し、憲法9条の改悪にも反対します。核兵器禁止条約に批准・署名し、核なき世界の実現を目指します。
エネルギー 脱原発を推進し、自然エネルギー100%の達成を目指します。
家族・子育て 出産に保険適用を導入し、子どもの医療費や保育料を無償化します。選択的夫婦別姓や同性婚の法制化も推進します。
